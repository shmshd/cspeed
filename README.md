## Endpoint reference
- `GET /todo` - Retrieves all todos
- `GET /todo/{id}` - Retrieves a single todo by `id`
- `POST /todo` - Create a new todo
- `PUT /todo/{id}` - Update a todo by `id`
- `DELETE /todo/{id}` - Delete a todo by `id`
